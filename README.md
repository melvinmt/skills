# Skills

A collection of Claude skills for specialized workflows, created by [Melvin Tercan](https://melvintercan.com/about).

## Available Skills

| Skill | Description |
|-------|-------------|
| [glm-47-prompting](skills/glm-47-prompting/) | Best practices for prompting GLM 4.7 |
| [rn-ios-extensions](skills/rn-ios-extensions/) | Build native iOS extensions from React Native |

## Installation

```bash
npx skills add melvinmt/skills
```

<details>
<summary>bun</summary>

```bash
bunx skills add melvinmt/skills
```

</details>

<details>
<summary>pnpm</summary>

```bash
pnpm dlx skills add melvinmt/skills
```

</details>

<details>
<summary>Claude Code</summary>

```
/plugin install <skill-name>
```

</details>

<details>
<summary>Manual installation</summary>

Copy the skill folder to your Claude skills directory.

</details>
