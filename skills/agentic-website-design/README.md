# Agentic Website Design Skill

Design and build production-quality websites using AI coding agents with full code control via static site generators.

## What's Included

- **Static Site Generator Workflow**: Astro-first approach with full LLM control over every line of code
- **Browser Feedback Loop**: Visual verification via Cursor Browser plugin or Claude Code Chrome integration
- **Skill-Enhanced Prompting**: Delegate to frontend-design, copywriting, and other skills for specialized work
- **Iterative Improvement**: High-bar polish loops with animations, micro-interactions, and responsive design
- **Prompting Patterns**: Ready-to-use prompts for every design phase

## Key Principles

| Principle | Description |
|-----------|-------------|
| Full code control | LLM generates all HTML, CSS, JS — no Framer/Webflow |
| Browser-in-the-loop | Screenshot and verify every change visually |
| Skill-enhanced prompting | Delegate to specialized skills for better results |
| Relentless iteration | Never accept the first output — push for 3+ rounds |
